ClosedFish is a cutting-edge chess engine designed to improve upon Stockfish by introducing specialized strategies for closed positions. It evaluates a "closedness coefficient" for each chessboard state, dynamically switching between the Stockfish engine and the ClosedFish algorithm for optimal performance.

Our Contributions

  • Closedness Coefficient Framework: Designed a "closedness coefficient" system to evaluate the positional state of the board and dynamically adapt strategies for closed positions.
  • Specialized Heuristics: Developed heuristics to assess pawn structures, piece mobility, and long-term planning for closed scenarios, addressing the limitations of traditional chess engines in such situations.
  • Alpha-Beta Pruning Optimization: Improved the efficiency of alpha-beta pruning to better handle the computational demands of deep closed position evaluations.
  • Neural Network Integration: Contributed to integrating an NNUE-based evaluation layer to enhance the engineâ€™s ability to make accurate decisions in both closed and open board states.
  • Dynamic Engine Switching: Helped implement a mechanism to seamlessly alternate between Stockfish and ClosedFish algorithms based on the closedness coefficient.
  • Team Collaboration: Collaborated with a 10-member team to integrate features, debug issues, and refine performance for various chess scenarios.

Building Instructions

First, clone the repository and ensure all submodules are cloned. This includes the Stockfish engine and vcpkg dependencies. 

git clone --recurse-submodules https://github.com/closedfish/closedfish.git
cd closedfish

Next, download the NNUE data file by navigating to the Stockfish directory and running the following command: 

cd external/stockfish/src
make net

(Optional) Compile the Stockfish project for a standard and portable configuration: 

make -j build ARCH=x86-64

Navigate back to the main project directory and configure the build using CMake: 

cd ../../..
cmake -B build -DCMAKE_BUILD_TYPE=Release

Finally, build the project to generate the executable: 

cmake --build build

Once the build is complete, locate the compiled binary, which should be available at: 

build/app/Debug/Executable.exe

Documentation

Comprehensive project documentation is generated using Doxygen and hosted online at: ClosedFish Documentation.

Getting Started

Once the project is built, you can use the compiled executable to analyze or play chess games. Example usage:

./Executable --analyze --game "example_game.pgn"
